# AutoInsight Dashboard

AutoInsight Dashboard is an automated business intelligence dashboard built with Python and Streamlit. It allows users to upload sales data in CSV format and automatically generates KPIs, charts, forecasts, and plain-English business insights.

## Project Objective

Companies rely on dashboards to monitor revenue, profit, customers, product performance, and regional trends. This project simulates that process by turning raw CSV data into useful business insights automatically.

## Features

- Upload CSV sales data
- Clean and prepare data automatically
- View executive KPIs
- Analyze sales, profit, region, and category performance
- Generate automated business insights
- Forecast next month's sales
- Use filters for business analysis

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
